*Hawk-Dove ESS Simulation*

A Python simulation of the Hawk-Dove Evolutionarily Stable Strategy (ESS) model from evolutionary game theory.

*Motivation*

While reading The Selfish Gene by Richard Dawkins, I came across the Hawk-Dove model and became fascinated by how simple interaction rules could shape the behavior of an entire population. Rather than only studying the theory, I decided to simulate the model in Python and observe its dynamics computationally.

*Features*
- Simulates repeated Hawk-Dove interactions
- Tracks population changes over time
- Exports simulation data to CSV
- Visualizes population dynamics using Matplotlib

*Built With*
- Python
- NumPy
- Matplotlib

*Results*

The simulation consistently converges to an approximately 1:2 ratio of Doves to Hawks, regardless of the initial population distribution or starting conditions under the implemented payoff rules. This demonstrates the emergence of a stable equilibrium through repeated interactions.
